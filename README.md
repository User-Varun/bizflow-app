# BizFlow App

BizFlow is a practical management tool for (Specific) B2B distributor businesses. Built with straightforward technology, it aims to simplify everyday business operations—so distributors can focus more on closing deals and less on juggling paperwork.

## Features

- Automatic Inventory management
- Simple, Modern Bill creation Experience
- access to all your bills at one place


## Tech Stack

- JavaScript / React + vite / CSS / supabase 

## Getting Started

Clone this repo, in terminal use npm run dev, and you’re set.
